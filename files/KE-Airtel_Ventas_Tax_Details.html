How every figure is calculated

Every formula below is taken directly from the PrimarySale sheet of the source workbook. All math runs on full, unrounded floating-point values — rounding to 2 decimal places happens only when a number is displayed on screen, never before it's used in the next calculation. That keeps long chains of totals accurate to the cent, exactly like the spreadsheet.

∞
Full precision, always.

Base prices in the Products sheet often run to 6+ decimal places (e.g. 0.7946027). The calculator carries that exact value through every step — Total Amount, Duty, VAT, Discount — and only rounds for the number you see in the table.
